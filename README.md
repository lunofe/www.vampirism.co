# www.vampirism.co
This repository contains the source content and generated output for a website built with Publii.

## Repository layout (minimal)
- `input/` - Publii sources (content, configuration, media, themes)
- `output/` - generated, publish-ready static site
- `functions/` / `wrangler.toml` - Cloudflare serverless functions
